GOAL:
Build a professional B2B website to showcase raw food products to foreign business clients.

TARGET AUDIENCE:
International business owners and importers.

- Frontend: HTML, Tailwind CSS, Vanilla JavaScript
- Backend: Node.js with Express
- Email: Nodemailer
- Data storage: JSON file (no database)

- Light theme with premium, minimal, corporate flow
- Export-industry style design
- Smooth layout, clean typography
- Fully responsive (mobile + desktop)
- Professional color palette (white, soft gray, subtle accent color)

1. Home
   - Hero section introducing the export business
   - Highlights of product quality and trust
   - Call-to-action buttons

2. About Us
   - Company overview
   - Mission & vision
   - Export focus

3. Products
   - Two categories:
     a) Regular Products
     b) Seasonal Exclusive Products
   - Product cards with name, description, category
   - Backend API to:
     - Add new products
     - Delete products

4. Licenses
   - Display export-related licenses and certifications
   - Clean showcase layout

5. Contact Us
   - Show company contact details
   - Contact form with:
     - Name
     - Email
     - Message

     - Send email to business owner using Nodemailer
     - Clear the form after successful submission

BACKEND REQUIREMENTS:
- Express server
- API routes for:

OTHER REQUIREMENTS:

OUTPUT EXPECTATION:
